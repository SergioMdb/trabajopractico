esto es solo idea mia para que sepa que se coloco con JavaScript
todo lo de java esta incorporado directo en las paginas sin necesidad de archivos JS externos, para facilitar mi edición

1. Fondo animado con partículas (login, index, contacto)
Función: Crea un fondo visual con partículas verdes en movimiento.

Objetivo: Estética gamer y moderna.

Tecnología: canvas o particles.js según la página.

Archivo: Código JavaScript directamente embebido en cada HTML o cargado con particles.min.js.

2. Formulario de contacto con envío AJAX (contacto.html)
Función: Envía el formulario sin recargar la página y muestra un mensaje de éxito.

Objetivo: Mejor experiencia de usuario y feedback inmediato.

Archivo: Script personalizado en contacto.html.

3. Mostrar/Ocultar soluciones (index.html)
Función: Botones para mostrar soluciones a problemas frecuentes y ofrecer contacto si no funcionan.

Objetivo: Asistencia interactiva al usuario con pasos guiados.

Archivo: Funciones mostrarSolucion() y mostrarContacto() en index.html.

4. Animación de logo (home y otras)
Función: Animación flotante del logo con @keyframes (usado junto a JS en estructura general).

Objetivo: Agregar dinamismo visual sutil.

Archivo: Parte del CSS, pero dependiente del entorno creado por el JS del fondo.
